함수 `route_to_42`의 프로토타입은 다음과 같습니다.

```javascript
const route_to_42 = () => {
  // TODO : return 42
}
```

- `route_to_42` 함수는 빈 객체를 매개변수로

- sunpark와 같이 프로젝트를 할 수 있는 카뎃의 인트라 아이디를 Set로 리턴해야 합니다.
